Import Instructions

Follow these guidelines to ensure successful import of your permission data.

  • Required Headers: Code, Name, Module, Action, Resource, Category, Sort Order, Status
  • Required Fields: Code, Name, Module, Action, and Resource are mandatory
  • Code: Must be unique and follow the format: Module.Action.Resource
  • Status: Must be "Active" or "Inactive" (case-insensitive)
  • Duplicates: Rows with existing codes will be automatically skipped
