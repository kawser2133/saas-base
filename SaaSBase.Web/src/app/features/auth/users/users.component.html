Import Instructions

Follow these guidelines to ensure successful import of your user data.

  • Required Headers: First Name, Last Name, Email, Department, Job Title, Location, Role, Phone Number, Status, Employee ID
  • Required Fields: First Name, Last Name, and Email are mandatory
  • Email: Must be unique and in valid format (e.g., user@example.com)
  • Status: Must be "Active" or "Inactive" (case-insensitive)
  • Duplicates: Rows with existing email addresses will be automatically skipped
  • Auto-Generated: Password and Full Name are automatically generated by the system
