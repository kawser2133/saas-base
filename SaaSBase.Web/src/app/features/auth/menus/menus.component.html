Import Instructions

Follow these guidelines to ensure successful import of your menu data.

  • Required Headers: Label, Route, Icon, Section, Parent Menu Label, Sort Order, Status
  • Required Fields: Label, Route, and Icon are mandatory
  • Section: Optional section name (e.g., SYSTEM ADMIN)
  • Parent Menu Label: Optional parent menu label for hierarchical menus
  • Status: Must be "Active" or "Inactive" (case-insensitive)
  • Duplicates: Rows with existing routes will be automatically skipped
